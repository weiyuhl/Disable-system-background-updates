== Description ==

用于WordPress后台禁止插件更新，禁止主题更新，禁止核心更新，隐藏更新通知，禁用管理员通知的WordPress插件

在WordPress中，禁止插件更新、禁止主题更新、禁止核心更新、隐藏更新通知和禁用管理员通知可以起到以下作用：

禁止插件更新：禁止插件更新可以确保您的插件保持在当前版本，防止更新可能引入的问题或不兼容性。这对于依赖特定版本插件的网站来说非常重要，以确保其功能正常运行。

禁止主题更新：禁止主题更新可以确保您的网站外观和样式保持一致，防止更新可能导致的布局变化或样式冲突。这对于使用自定义主题或对外观有严格要求的网站来说很有用。

禁止核心更新：禁止核心更新可以防止WordPress核心文件被更新，确保您的网站保持在特定版本。这对于那些依赖特定版本核心文件的网站来说很重要，以确保插件和主题的兼容性，并避免潜在的安全风险。

隐藏更新通知：隐藏更新通知可以减少管理员或用户对更新的干扰，使他们无需关注和处理更新的提示。这对于那些不希望频繁更新或希望保留控制权的网站管理员来说很有用。

禁用管理员通知：禁用管理员通知可以隐藏管理员通知，如更新提示、插件活动消息等。这可以减少干扰，提高管理员的工作效率，并确保他们只接收到必要的通知。

== Installation ==

1. 把 Disable-updates 文件夹上传到 /wp-content/plugins/ 目录下
2. 在后台插件列表中激活 禁止后台更新
3. 开始使用吧~
